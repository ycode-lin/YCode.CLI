cli 门户 / 编排 / agent runtime

把代码操作成一套正在运行的终端系统，而不是静态聊天框。

YCode CLI 面向真实仓库工作流构建，把 explore、plan、code 拆给不同子代理，通过 MCP 扩展操作面，并用记忆层和技能包维持长链路任务的一致性。

bootstrap dotnet tool install --global YCode.CLI
执行模型 Explore / Plan / Code

子代理让上下文更小，推理和决策也更清晰。
