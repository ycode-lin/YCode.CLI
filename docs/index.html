Repository-native coding agent

YCode CLI

A .NET 10 command line agent built for real repository work. YCode combines subagents, MCP tools, memory layers, and skill loading into one terminal-first flow.

Install from NuGet View Source
Install dotnet tool install --global YCode.CLI
Why it stands out
  • Subagents split exploration, planning, and code execution.
  • MCP tools extend filesystem and system capabilities.
  • Memory and skills make repeated work faster and more structured.
